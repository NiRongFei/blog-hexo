使用 vw 定制 rem 自适应布局

要点：移动端使用 rem 布局需要通过 JS 设置不同屏幕宽高比的 font-size，结合 vw 单位和 calc() 可脱离 JS 的控制。

场景：rem 页面布局(不兼容低版本移动端系统)。

兼容：vw、calc()
