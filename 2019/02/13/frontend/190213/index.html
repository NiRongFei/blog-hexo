配置雪碧图

默认 src/assets/icons 中存放需要生成雪碧图的 PNG 格式图片。首次运行 npm run serve/build 会生成雪碧图，并在根目录生成 icons.json 文件。再次运行命令时，会对比 icons 目录下图片文件与 icons.json 的匹配关系，确定是否需要再次执行 webpack-spritesmith 插件。
